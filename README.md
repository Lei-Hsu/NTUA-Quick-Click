## 作品說明：

大學時，每個學期結束都需要填寫教學評量問卷。<br>
如果有些同學一學期選了很多學分，需要填寫的問卷量也會大幅增加。<br>
鑑於問卷歷年來的形式、問題以及答案都相同，所以做了一個自動填寫問卷的小應用程式且上架到google商店。
